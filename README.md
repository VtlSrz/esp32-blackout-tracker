*** Проєкт на базі плати esp32 c3 super mini та плати зарядки аккумулятора з step booster 5v 1A ***

ВАЖЛИВО! Для роботи потрібен доступ до мережі WiFi 2.4G

Проект потребує мінімальну кількість електронних компонентів та мінімальних навичок паяння. 
При відключенні та включенні електроенергії відправляє повідомлення в телеграм бота. 
Дозволяє моніторити фактичну наявність та відсутність електроєнергії незалежно від вашого перебування.

*** Принцип роботи ****

При включенні піднімає WiFi AP, приєднавшись та відвідавши ip - 192.168.4.1 отримуємо доступ до веб сторінки керування платою. 
На вебсторінці потрібно заповнити поля з існуючим SSID (логін) та Password вашого роутера після чого плата переходить в режим WiFi STA та поводить себе як звичайни пристрій. 
Для переходу на сторрінку налаштувань у браузері потрібно ввести url адресу - light-tracker.local (потрібно щоб пристрій був у тій самій WiFi мережі що і плата), далі ввести token та chat id вашого бота після чого
обрати необхідні вам налаштування, далі все працює в автономному режимі.
